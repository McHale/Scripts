    About

    This is an open repository for OpenEUO scripts for Ultima Online: Excelsior Free Shard. The goal of this project is to allow our community programmers to have access to a repository of scripts, especially ones that need to be cleaned, fix bugs, add more features, or implement a user interface/gui. This project also enables programmers to add their own scripts into this common repository. You don't have to have a GitHub login to be able to download this repository for personal use of the scripts.

    In order to add to this repository or make changes to files in this repository, you must have a GitHub Username. Contact me on the Excelsior Forums. Be sure to include in a Private Message, your GitHub Username to be added to the list of collaborators, as well as your in-game name.

    Getting Started With Git

    Git is a wonderful way to track and work with others on the same code base. There are many resources at there to help you get started with Git, which I'll include later on.
